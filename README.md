# Vedprakash Wellness

A reactive wellness application built with React and TypeScript.

## Project Structure

```
vedprakash_wellness/
├── my-reactive-app/    # Main React application
│   ├── public/         # Static assets
│   ├── src/            # Application source code
│   ├── package.json    # Dependencies
│   └── README.md       # App-specific documentation
└── .git/               # Version control
```

## Getting Started

Navigate to the `my-reactive-app` directory and follow the instructions in its [README.md](./my-reactive-app/README.md).

```bash
cd my-reactive-app
npm install
npm start
```
